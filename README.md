# STRATA

**STRATA は、対話のための条件設定インターフェースである。**  
説明・評価・最適化を行わない。
STRATA は、問いや状況に含まれる条件を削らずに並置し、  
同一粒度の複数出力を返し、  
そのあいだに生じるズレを位置として示す。
STRATA は以下を行わない。

- 解釈
- 意味づけ
- 評価・判断
- 推奨
- 最適化
- 意思決定
- 理論・思想への参照

STRATA は理論ではない。外部フレームワークを参照しない。

このリポジトリは **docs-only** であり、  
実装、運用、価格、事例は含まれない。

## English
**STRATA is a condition-setting interface for dialogue.**  
It does not explain, evaluate, or optimize.

STRATA places conditions side by side without reduction,  
returns multiple outputs at the same level of granularity,  
and shows misalignments as positions.

STRATA does not perform:
- Interpretation
- Meaning-making
- Evaluation or judgment
- Recommendation
- Optimization
- Decision-making
- Reference to theories or philosophies

STRATA is not a theory.  
It does not reference any external framework.

This repository is **docs-only**.  
No implementation, operation, pricing, or case materials are included.
